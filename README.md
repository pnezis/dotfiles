## Installation

1. `git clone http://github.com/pnezis/dotfiles.git ~/dotfiles`
2. `cd ~/dotfiles`
3. `setup.sh` to create symbolic links and install dependencies.
4. Run Vim and type `:BundleInstall` to install the plugins with Vundle.
